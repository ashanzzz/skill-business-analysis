# 商业分析Skill方法论

> 四轮验证 → 十二维分析 → 结构化输出 → 报告交付

## 核心框架

### 四轮验证
1. **多源数据验证** - 3个独立来源交叉验证
2. **逻辑自洽验证** - 供给=需求、份额≤100%、LTV>3×CAC
3. **案例类比验证** - 同类市场案例对比
4. **时间序列验证** - 历史CAGR与预测一致性

### 十二维分析
| 维度 | 内容 |
|------|------|
| D1 | 市场规模（TAM/SAM/SOM） |
| D2 | 区域市场差异（8大区域） |
| D3 | 竞争格局（集中度+壁垒+颠覆风险） |
| D4 | 消费者行为（画像+决策） |
| D5 | 监管环境（8大领域对比） |
| D6 | 供应链分析（风险+建议） |
| D7 | 技术成熟度（Gartner位置） |
| D8 | 人才分布（成本对比） |
| D9 | 宏观风险（风险矩阵） |
| D10 | 财务建模（LTV/CAC/估值） |
| D11 | GTM策略（进入模式） |
| D12 | 投资决策（NPV/IRR） |

## 使用方法

1. 用户说"分析XX"
2. AI确认核心4问（分析什么/地理/目的/客户）
3. 执行四轮验证 + 十二维分析
4. 输出结构化报告
